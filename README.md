# dnssdk
The DNS API Go package for Cloudflare, Alidns, DNSPod and PQDNS.

# Services

## Domain
- DomainList 域名列表
- DomainAdd 域名添加
- DomainDelete 域名删除

## Record
- RecordList 记录列表
- RecordAdd 记录新增
- RecordUpdate 记录修改
- RecordDelete 记录删除
- RecordEnable 记录启用
- RecordDisable 记录暂停
- RecordStatusSupported 记录支持状态？
